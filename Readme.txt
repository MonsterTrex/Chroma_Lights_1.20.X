Chroma Lights by MonsterTrex

Have you ever wanted more colours of lights for Minecraft? Maybe you use a shader which uses coloured lights to create beautiful scenery. Now you can mix and match various types of coloured lights using this mod!

Glowglass is a new coloured light block that is made with glass surrounded by 4 glowstone dust. After any of the light blocks are made, you can colour them like any other coloured variant in the game - place the block with the dye and tadaa! Beautiful coloured lights!
